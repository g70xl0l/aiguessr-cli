Как использовать

1. Подготовка

* Получите бесплатный API ключ в Google AI Studio.

* Установите необходимые библиотеки, открыв терминал (или командную строку) и введя:

` pip install pillow requests `


2. Настройка и Запуск

* Запустите скрипт командой:

` python GeoAI.py `


В открывшемся окне программы найдите текстовое поле «Ключ Gemini API:» (в самом верху) и вставьте туда свой скопированный ключ.

💡 Лайфхак (чтобы не вводить ключ каждый раз): Откройте файл geoguessr_helper.py в любом текстовом редакторе (Блокнот, VS Code и т.д.), найдите на 10-й строке переменную API_KEY = "" и вставьте ваш ключ между кавычками. Например: API_KEY = "AIzaSy...". После этого сохраните файл, и ключ будет загружаться автоматически!

3. Играем!

Откройте раунд в GeoGuessr в браузере или в ПК версии игры из Steam.

В приложении нажмите «📸 1. Выделить область» и обведите саму панораму игры (постарайтесь захватить максимум улицы и деталей, но без интерфейса браузера).

Нажмите зеленую кнопку «🔍 2. ГДЕ Я?» и подождите пару секунд.

Бот проанализирует картинку и выдаст в текстовое окно страну, предполагаемый город/регион и список найденных улик (мет)!
